issue-tracker
=============

Repository for the issue tracker application.

#### Technology Stack

* Angular.js 1.2.16
* Angular.js Bootstrap Directives 0.10
* Bootstrap 3.1.1
* Cubism.js 1.0.0
* D3.js 3.4.1
* Font Awesome 4.0.3
* jQuery 2.1.0
* Node.js v0.10.26
  * Express

#### Starting Server
In the application folder type: `node server.js`
